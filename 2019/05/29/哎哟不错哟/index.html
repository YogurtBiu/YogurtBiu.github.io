Command line instructions

Git global setup
git config –global user.name “YogurtBiu”
git config –global user.email “1129367355@qq.com“

Create a new repository
git clone git@gitlab.com:YogurtBiu/Anniversary.git
cd Anniversary
touch README.md
git add README.md
git commit -m “add README”
git push -u origin master

Existing folder
cd existing_folder
git init
git remote add origin git@gitlab.com:YogurtBiu/Anniversary.git
git add .
git commit -m “Initial commit”
git push -u origin master

Existing Git repository
cd existing_repo
git remote rename origin old-origin
git remote add origin git@gitlab.com:YogurtBiu/Anniversary.git
git push -u origin –all
git push -u origin –tags
